Anthony Seldon Portfolio Website
Description

This project is a personal portfolio website created using HTML5 and CSS3. The website introduces visitors to who I am, highlights projects I have completed, and provides a contact form for communication. The purpose of this project is to demonstrate my understanding of web development fundamentals, semantic HTML structure, and responsive web design principles.

Features
Personal introduction and About Me section
Navigation menu for easy page access
Project showcase displayed in a table format
Contact form for visitor inquiries
Semantic HTML5 structure
Responsive design using CSS
Technologies Used
HTML5
CSS3

Website Structure
Home Page

The homepage welcomes visitors and provides navigation to different sections of the portfolio.

About Me

This section contains information about my background and current web development learning journey.

Projects

The projects section showcases completed assignments and portfolio work with descriptions of each project.

Contact

The contact section contains a form that allows visitors to submit their name, email address, subject, and message.
